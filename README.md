# focsalbosc
Visor incendis forestals Catalunya 1986-2016

## Deploy
El deploy automatic es fa en una màquina amb les següents dependencies:
 - Apache o nginx
 - Nodejs
```
mkdir dist
npm install
npm build-min
mv ./* /var/www/html/
```

### Crear directori web *"focalbosc"*

### Copiar

**index.html** i el directori **/dist** dins directori web
